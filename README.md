# 🦑 Kraken Motorsports Website

Pre-launch website for Kraken Motorsports - a VR racing rig experience coming to a community lounge.

## 🎯 Purpose

This website serves as a pre-launch hub to:
- Build hype for the Kraken Rig before it physically exists
- Establish the Kraken Motorsports brand with a strong visual identity
- Explain the concept clearly to potential users and supporters
- Collect early interest through social media
- Prepare for Founders Pass sales
- Showcase the vision with mockups and UI previews

## 🚀 Getting Started

### View Locally
1. Open `index.html` in any modern web browser
2. No server or build process required!

### Customize
- **Change Launch Date**: Edit `script.js` line 9 to set target date
- **Update Social Links**: Replace `#` in `index.html` with actual URLs
- **Add Content**: Modify text and images in `index.html`
- **Adjust Colors**: Update CSS variables in `:root` section of `styles.css`
