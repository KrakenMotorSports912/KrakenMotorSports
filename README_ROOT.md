## 🎯 What You Have

✅ **Live Leaderboard** - Real-time lap time tracking with Supabase  
✅ **Admin Dashboard** - Manage events, discounts, and submissions  
✅ **Authentication** - User accounts with admin roles  
✅ **Founders Pass** - Limited pass system with tracking  
✅ **Event System** - Create and manage racing events  
✅ **Discount Codes** - Promotional system  
✅ **Multi-game Support** - Assetto Corsa, F1 2025, Forza, etc.  
✅ **Real-time Updates** - Supabase Realtime WebSockets  
✅ **Responsive Design** - Works on mobile, tablet, desktop  
✅ **Dark-Cyan Theme** - Professional Kraken branding  

---

## 💻 Tech Stack

- **Framework:** Next.js 14 (React, TypeScript)
- **Styling:** Tailwind CSS
- **Database:** Supabase (PostgreSQL + Realtime)
- **Auth:** Supabase Auth
- **Hosting:** Vercel (recommended)

## ⚡ Commands

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start

# Check for errors
npm run lint
```

Open [http://localhost:3000](http://localhost:3000) after running `npm run dev`
