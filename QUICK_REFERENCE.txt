═══════════════════════════════════════════════════════════════
  ✅ SETUP CHECKLIST (45 minutes total)
═══════════════════════════════════════════════════════════════

[ ] 1. INSTALL PACKAGES (5 min)
    Command: npm install
    
[ ] 2. CREATE SUPABASE ACCOUNT (5 min)
    Website: https://supabase.com
    Create new project: "kraken-motorsports"
    
[ ] 3. RUN DATABASE SCHEMA (10 min)
    Open: SUPABASE_SETUP.md
    Copy SQL → Paste in Supabase SQL Editor → Run
    
[ ] 4. GET API KEYS (2 min)
    Supabase → Project Settings → API
    Copy: Project URL, anon key, service_role key
    
[ ] 5. CREATE .ENV.LOCAL (5 min)
    Copy: .env.local.example to .env.local
    Paste your Supabase keys
    
[ ] 6. START DEV SERVER (1 min)
    Command: npm run dev
    Open: http://localhost:3000
    
[ ] 7. CREATE ADMIN ACCOUNT (5 min)
    Sign up on your site
    Copy your user ID from Supabase
    Run SQL to set is_admin = true
    
[ ] 8. TEST ADMIN DASHBOARD (2 min)
    Visit: http://localhost:3000/admin
    Verify you can see the dashboard
    
[ ] 9. PUSH TO GITHUB (5 min)
    Commands in INSTALLATION_GUIDE.md
    
[ ] 10. DEPLOY TO VERCEL (10 min)
    Website: https://vercel.com
    Import GitHub repo
    Add environment variables
    Deploy!
