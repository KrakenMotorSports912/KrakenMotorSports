## Image Specifications

- **Logo**: PNG with transparency, minimum 1000px wide
- **Hero Images**: 1920x1080px or larger
- **Mockups**: High resolution, 1200px+ wide
- **Social Images**: Follow platform specifications
- **Favicon**: 512x512px, will be resized for browser use
